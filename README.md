## Dependencies

* Ruby v2.7.0

* Rails v5.2.3

## First time setup

Run the installation script `install.sh`

`$ ./install.sh`

This should install all necessary dependencies to the system, and runs `yarn install` and `bundle install` to get it everything setup. 

## Running the project

`$ rails server` or `$ rails s`
